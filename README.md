# Glass Bead Renderer

A software renderer just for a pure fun

## Building

### macOS

Not that you should have a Developer Tools installed

```
$ brew install sdl2 sdl2_ttf
$ git clone git@github.com:meatmachine/glass-bead-renderer.git
$ cd glass-bead-renderer
$ mkdir _build
$ make gbr
$ ./gbr
```
